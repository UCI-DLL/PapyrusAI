# PapyrusAI Frontend

## Overview
PapyrusAI is a web-based application designed to help students with research and learning using AI.
Learn more about PapyrusAI and the Digital Learning Lab [here](https://www.genaied.org/papyrusai.html)

## Live Version
See PapyrusAI in action at [papyrusai.org](https://www.papyrusai.org/)

## Setup
create a .env file linking to your backend. 

```
REACT_APP_API_URL=
REACT_APP_WEBSOCKET_URL=
REACT_APP_LOGIN_URL=
REACT_APP_INSTRUCTOR=
REACT_APP_ADMIN=
```

### `npm install`
install npm packages

### `npm start`
Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

This repository contains the **frontend React application**.

> Note: The backend and infrastructure are not included in this repository.
> This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).
